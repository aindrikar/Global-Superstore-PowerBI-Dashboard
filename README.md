# 📊 Global Superstore Sales Analysis Dashboard (Power BI)

## Overview
This project focuses on analyzing retail sales data using Power BI to understand business performance, profit trends, and regional distribution. The dashboard provides interactive visualizations to support data-driven decision-making.

##  Data Preparation
- Cleaned and preprocessed the dataset  
- Removed null and duplicate values  
- Converted date columns to proper format  
- Created calculated fields such as profit margin and time-based attributes  

##  Dashboard Features
- KPI Cards (Total Sales, Profit, Orders, Profit Margin)  
- Sales Trend Over Time  
- Sales by Region  
- Profit by Category  
- Sales by Segment  
- Map Visualization for geographical analysis  
- Top Products Analysis  

##  Key Insights
- Identified top-performing regions and product categories  
- Observed trends and seasonal variations in sales  
- Detected differences between high sales and high profit categories  
- Highlighted areas for improving profitability  

##  Tools Used
- Power BI  
- Excel / CSV Dataset
